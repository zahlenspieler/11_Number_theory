normal order


Let f⁢(n) and F⁢(n) be functions from ℤ+→ℝ. We say that f⁢(n) has normal order F⁢(n) if for each ϵ>0 the set

A⁢(ϵ)={n∈ℤ+:(1-ϵ)⁢F⁢(n)<f⁢(n)<(1+ϵ)⁢F⁢(n)}

has the property that d¯⁢(A⁢(ϵ))=1. Equivalently, if B⁢(ϵ)=ℤ+\A⁢(ϵ), then d¯⁢(B⁢(ϵ))=0. (Note that d¯⁢(X) denotes the lower asymptotic density of X).

We say that f has average order F if

∑j=1nf⁢(j)∼∑j=1nF⁢(j)
