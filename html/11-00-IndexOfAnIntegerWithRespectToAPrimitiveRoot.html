index of an integer with respect to a primitive root


Definition.

Let m>1 be an integer such that the integer g is a primitive root for m. Suppose a is another integer relatively prime to g. The index of a (to base g) is the smallest positive integer n such that gn≡amodm, and it is denoted by ind⁡a or indg⁡a.

If m has a primitive root the index with respect to a primitive root is a very useful tool to solve polynomial congruences modulo m.
