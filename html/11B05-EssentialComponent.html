essential component


If A is a set of nonnegative integers such that

σ⁢(A+B)>σ⁢B (1)

for every set B with Schnirelmann density 0<σ⁢B<1, then A is an essential component.

Erdős proved that every http://planetmath.org/node/3831basis is an essential component. In fact he proved that

σ⁢(A+B)≥σ⁢B+12⁢h⁢(1-σ⁢B)⁢σ⁢B,

where h denotes the http://planetmath.org/node/3831order of A.

Plünnecke improved that to

σ⁢(A+B)≥σ⁢B1-1/h.

There are non-basic essential components. Linnik constructed non-basic essential component for which A⁢(n)=O⁢(nϵ) for every ϵ>0.
