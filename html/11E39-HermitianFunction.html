Hermitian function


A Hermitian function is complex valued function whose real part is even and imaginary part is odd.

1 Definition

Let f⁢(x) be a complex valued function of a real variable x. f⁢(x) is a Hermitian function if f⁢(-x)=f⁢(x)¯ for all x.
