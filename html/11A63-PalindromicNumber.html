palindromic number


An integer n that in a given base b is a palindrome. Given n being k digits dx long in base b, its value being

n=∑i=0k-1dk⁢bi

then if the equalities dk=d1, dk-1=d2, etc., hold, then n is a palindromic number. There are infinitely many palindromic numbers in any given base.
