cases when minus one is a quadratic residue


Theorem.

Let p be an odd prime. Then -1 is a quadratic residue modulo p if and only if p≡1mod4.

Proof.

Let p be an odd prime. Notice that p is congruent to either 1 or 3 modulo 4. By the definition of the Legendre symbol, we need to verify that (-1p)=1 if and only if p≡1mod4. By Euler’s criterion

(-1p)≡(-1)(p-1)/2modp.

Finally, note that the integer p-12 is even if p≡1mod4 and odd if p≡3mod4. ∎
