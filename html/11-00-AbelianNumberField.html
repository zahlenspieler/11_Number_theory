abelian number field


Definition 1.

An abelian number field is a number field K such that K/Q is an abelian extension, i.e. K/Q is Galois and Gal⁡(K/Q) is an abelian group.

The abelian number fields are classified by the Kronecker-Weber Theorem.

Definition 2.

A cyclic number field is an (abelian) number field K such that K/Q is a Galois extension and Gal⁡(K/Q) is a finite cyclic group (therefore abelian).
